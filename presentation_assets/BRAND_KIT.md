# FSW Presentation Brand Kit for Canva

Use this guide to set up your "Brand Kit" in Canva or manually apply these styles to your presentation.

## 1. Color Palette (Hex Codes)

Copy and paste these codes into Canva's color picker.

### Primary Colors
*   **FSW Blue (Primary):** `#128ECD`
*   **FSW Green (Accent):** `#00843B`
*   **FSW Navy (Deep):** `#1B4E91`

### Backgrounds & Neutrals
*   **Pure Black (Background):** `#000000`
*   **Glass Surface (Panel):** `#141E3C` (Set transparency to ~40% in Canva)
*   **Border/Divider:** `#FFFFFF` (Set transparency to ~8%)
*   **Text (White):** `#FFFFFF`
*   **Text (Muted):** `#A3A3A3`

---

## 2. Typography

Canva has a wide font library. The FSW platform uses **Inter**, which is available in Canva.

*   **Headings (Title Slide):** Inter Bold (700) - Size 60pt+
*   **Subheadings:** Inter Semi-Bold (600) - Size 32pt+
*   **Body Text:** Inter Regular (400) - Size 18pt+
*   **Buttons/Labels:** Inter Medium (500)

*Tip: If you want a more "Cinematic" header, try using "Inter Tight" or increase the letter-spacing slightly for uppercase text.*

---

## 3. Visual Style Guide

To recreate the "Glassmorphism" look in Canva:

1.  **Shapes:** Use the "Rounded Square" shape.
2.  **Color:** Set color to `#141E3C`.
3.  **Transparency:** Lower the opacity to `40%`.
4.  **Border:** Add a thin border (Weight: 1) in White `#FFFFFF` and set the border transparency to `20%`.
5.  **Blur:** Search for "Blur" in Elements and place a soft colored blur behind the glass shape to make it pop.
